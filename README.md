An AI-driven web application that helps developers and designers instantly generate tailored UI kits. By selecting an industry and a desired "vibe," the built-in Machine Learning model predicts the optimal layout, color palette, and typography. 

Users can preview the design live, generate expert UX suggestions using Google's Gemini AI, and export the complete, responsive code (HTML, Tailwind CSS, and Vanilla JavaScript) as a ZIP file.

* **Smart Generation:** Uses a trained ML model to suggest layouts and colors based on 20+ specific industries.

* **AI UX Assistant:** Integrates the Gemini API to provide actionable UX advice for the generated layout.
* **Dynamic Imagery:** Automatically fetches relevant, high-quality placeholder images via the Unsplash API.
* **One-Click Export:** View the code in the browser or download the complete `index.html`, `style.css`, and `script.js` files bundled in a `.zip`.
